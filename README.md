# ForeSite AI — Project

## Project info

This repository contains the ForeSite AI website built with Vite + React.

## How can I edit this code?

Use your preferred IDE to edit files locally. Basic steps:

1. Clone the repository.
2. Install dependencies: `npm i`.
3. Start dev server: `npm run dev`.

If you prefer working in the cloud you can use GitHub Codespaces or other online IDEs.

## What technologies are used for this project?

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

Follow your preferred deployment workflow (Vercel, Netlify, Docker, etc.) — build output will be produced by `npm run build`.
